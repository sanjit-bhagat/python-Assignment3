# Factorial and Math Module Programs

## Task 1: Factorial Calculation
This program calculates the factorial of a number using a recursive function.

## Task 2: Math Module Operations
This program takes a number from the user and calculates:
- Square root
- Natural logarithm
- Sine value (in radians)

## How to Run
1. Clone the repository
2. Run the Python files using:
   python factorial.py
   python math_operations.py
